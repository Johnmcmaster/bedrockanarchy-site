Guide

How to join BedrockAnarchy.

Use this address and port to join quickly from Bedrock.

Bedrock Edition

Server details

Server address: mc.bedrockanarchy.org

Port: 19132

  1. Open Minecraft Bedrock.
  2. Go to Servers and choose Add Server.
  3. Enter mc.bedrockanarchy.org and port 19132.

Troubleshooting

If you cannot connect

  • Check you are on Bedrock edition, not Java.
  • Re-enter the address as mc.bedrockanarchy.org.
  • Confirm port is exactly 19132.
